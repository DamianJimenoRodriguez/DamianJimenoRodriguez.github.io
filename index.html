Lady Bug Parkour

By using MQTT we made an interactive Parkour game.

We use custom led panels spread over a big room. Then, the clients are given watches with Bluetooth beacons. Said beacons can interact with the custom panels. Additionally, we made a Unity application that is connected to the MQTT broker. This application can detect the interactions between the beacons and the led panels and it can send commands to the broker. Thought this application it is possible to play multiple games.

Used stack:

  • Unity
  • C#
  • MQTT
  • Git
